# Student Search Web Application (Java Servlets)

## 🚀 Project Description
This is a simple Java Servlet-based web application where users can search student details using their ID.

The application fetches data from a MySQL database and displays it on the web page.

---

## 💡 Features
- Search student by ID
- Retrieve data using doGet()
- JDBC integration with MySQL
- Simple and clean UI
- Displays Name and Age dynamically

---

## 🛠️ Tech Stack
- Java Servlets
- JDBC
- MySQL
- HTML, CSS
- Apache Tomcat

---

## 🔄 Workflow
User enters ID → Servlet (doGet) → Database → Result displayed

## ⚙️ How to Run

1. Clone the repository
2. Import project into IDE (Eclipse/IntelliJ)
3. Configure MySQL database
4. Update DB credentials in servlet
5. Deploy on Apache Tomcat
6. Run index.html
